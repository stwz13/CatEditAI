# CatEdit AI

CatEdit AI - офлайн-фоторедактор для ОС Аврора. Приложение позволяет выбрать
фотографию из галереи или файловой системы, применить локальные инструменты
обработки, отменить последнее действие, сбросить результат к оригиналу и
экспортировать обработанное изображение обратно в галерею.

Главная идея продукта - приватность: фотографии обрабатываются на устройстве и
не отправляются на внешний сервер.

## Возможности

- Импорт изображений из системной галереи.
- Импорт изображений из файловой системы.
- Отображение оригинала и текущей рабочей копии.
- Замена фона с помощью OpenCV GrabCut.
- **Нейросетевая стилизация** (Fast Neural Style / Mosaic) через ncnn, офлайн на устройстве.
- Автоматическое улучшение яркости, контраста и цветового баланса.
- Простые локальные стили: черно-белый фильтр и сепия.
- Последовательное применение операций к рабочей копии.
- Отмена последнего действия.
- Сброс изображения к оригиналу.
- Экспорт результата в галерею.

## Технологии

- Нативное приложение для ОС Аврора.
- C++ и QML.
- Qt 5 и компоненты Sailfish Silica.
- Проект qmake.
- RPM-упаковка через Build Engine из Aurora SDK.
- OpenCV `core` и `imgproc` для обработки изображений.
- [ncnn](https://github.com/Tencent/ncnn) для нейросетевой стилизации (модель Mosaic, ~3.2 MB).

## Сборка ncnn (один раз)

См. [third_party/ncnn/README.md](third_party/ncnn/README.md). Кратко: в Build Engine клонировать исходники ncnn и запустить `build_ncnn.sh` с именем вашей папки сборки.

## Архитектура

`ImageProcessor` - контроллер, доступный из QML. Он хранит состояние интерфейса:
`busy`, `hasImage`, `canUndo`, `originalSource`, `resultSource`, а также
предоставляет методы для импорта, обработки, отмены, сброса и экспорта
изображений.

`ProcessingWorker` выполняет обработку изображений в отдельном `QThread`, чтобы
интерфейс QML не зависал во время тяжелых операций.

`ResultImageProvider` передает сгенерированные изображения в QML через
`image://result/...`. Изображения защищены мьютексом, потому что запросы к image
provider могут выполняться не из GUI-потока.

## Разрешения

В desktop-файле запрошены разрешения:

```ini
Permissions=Pictures;UserDirs
```

`Pictures` нужен для работы с галереей и экспорта результата. `UserDirs`
оставлен, потому что приложение также поддерживает выбор изображения через
файловый picker.
